# smart-eater
Smart-Eater adalah platform edukasi digital berbasis web yang dirancang khusus untuk remaja guna meningkatkan kesadaran akan pola makan sehat, gizi seimbang, dan bahaya konsumsi fast food berlebihan. Proyek ini bertujuan untuk membantu remaja memahami risiko obesitas serta memberikan panduan praktis gaya hidup sehat dengan pendekatan yang mudah dipahami.

🚀 Fitur Utama
Edukasi Gizi & Obesitas: Materi informatif tentang gizi seimbang dan pentingnya menjaga berat badan ideal bagi remaja.

Kesadaran Fast Food: Penjelasan mengenai dampak buruk fast food menggunakan pendekatan edukatif (prinsip 80/20).

Kuis Interaktif: Evaluasi pemahaman melalui kuis gizi yang menarik.

Integrasi Game: Hubungan langsung ke tantangan Kahoot untuk pengalaman belajar yang lebih seru.

🛠 Teknologi yang Digunakan
HTML5 - Struktur konten web.

CSS3 - Styling dan desain antarmuka responsif.

JavaScript - Logika kuis interaktif dan fungsionalitas web.

Vercel - Platform deployment untuk hosting aplikasi.
